## 🎯 这是一个完整的 HarmonyOS 贪吃蛇游戏项目

### ✨ 项目特色

- 🎮 **完整游戏** - 经典贪吃蛇玩法，流畅的游戏体验
- 🎨 **精美UI** - 现代化深色主题设计
- 📱 **HarmonyOS** - 基于 HarmonyOS 6.0.2(22) 开发
- 📚 **完整文档** - 11个文档文件，详尽的项目说明
- 🤝 **开源友好** - MIT许可证，完善的贡献指南

### 🚀 快速开始

#### 1️⃣ 打开项目
```
DevEco Studio → File → Open → 选择项目目录
```

#### 2️⃣ 运行项目
```
连接设备或启动模拟器 → 点击运行按钮
```

#### 3️⃣ 开始游戏
```
使用方向按钮控制蛇的移动
吃到红色食物增加长度和分数
避免撞墙和撞到自己
```

### 💡 自定义配置

#### 修改游戏速度
```typescript
// 在 entry/src/main/ets/pages/Index.ets 中
private gameSpeed: number = 150; // 改为 100 更快，200 更慢
```

#### 修改网格大小
```typescript
// 在 entry/src/main/ets/model/SnakeGame.ets 中
private gridWidth: number = 20;  // 宽度
private gridHeight: number = 30; // 高度
```

#### 修改颜色主题
```typescript
// 在 entry/src/main/ets/pages/Index.ets 中
.backgroundColor('#1a1a2e')  // 背景色
.backgroundColor('#e94560')  // 强调色
```
